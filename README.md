Pinyin
=========
	把中文转换成拼音的JS，不依赖其他JS文件，可直接使用
How to use
----------

	 1）将汉字翻译为拼音，其中每一个字的首字母大写：	
	 pinyin.getFullChars(this.value);
	 2）将每一个字的拼音的首字母提取出来，是大写的形式：
	 pinyin.getCamelChars(this.value);
